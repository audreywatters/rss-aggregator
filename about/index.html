About Project

This is a project born out of the indie ed-tech discussions that occurred recently at Davidson College in North Carolina.

It is about exploring a simple client side RSS aggregator that runs 100% on Github, using the API, and underlying Github repo as storage.

While this project is functional, and you can fork and run in your repo, it is also meant to be simple, and lacking many advanced features.

You can use the Github Issues for this project to get support.
